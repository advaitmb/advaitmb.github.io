# Advait Bhat CV

LaTeX source for the CV published from this repo.

## Build

From this directory:

```sh
lualatex cv.tex
```

Run it twice if the table of contents or references need to settle.

## Notes

- `cv.tex` is the entry point.
- `sections/` contains the authored content.
- Keep publication titles and statuses aligned with the website homepage and project pages.
